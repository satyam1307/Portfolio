Steganography


Steganography means embedding data into an image such that the image doesn’t get corrupted. This concept can be used to transfer data over a network as the human eye cannot detect the data hidden inside an image.


So, the data can be embedded inside an image and the image can be transferred and later on the data can be extracted from the image.
